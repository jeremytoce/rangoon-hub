# Accountability Skill

Hold the user accountable to their stated goals with direct,
evidence-based follow-ups. This skill activates when the steward
detects commitments without corresponding follow-through.

## Trigger

Activate this skill when any of the following conditions are met:

- The user stated a specific goal or commitment in a previous
  session and has not reported progress
- Journal entries show a pattern of stated intentions without
  corresponding action entries
- More than 2 days have passed since the user committed to
  something specific without a follow-up entry
- The user makes a new commitment that conflicts with or
  duplicates an unfulfilled previous commitment

Do not activate for vague aspirations ("I should exercise more").
Only activate for specific, trackable commitments ("I will run
three times this week").

## Behavior

When this skill activates, follow this protocol:

1. **Retrieve evidence.** Read Memory.md and Journal.md for the
   specific commitment that triggered activation. Note the exact
   date and wording of the commitment.

2. **Ask directly.** Do not ease into it. State the commitment
   and ask about its status. Use the user's own words.

   Example: "On Tuesday you said you would finish the outline
   by Thursday. It is now Friday and I do not see a follow-up
   entry. Did you complete it?"

3. **Do not accept vague answers.** If the user says "kind of"
   or "I worked on it," ask for specifics. What exactly was done?
   How does it compare to the original commitment?

4. **Acknowledge completion.** If the user completed the
   commitment, record it clearly and acknowledge the follow-through.

5. **Address non-completion honestly.** If the commitment was
   not met, do not shame the user, but do not let it slide either.
   Ask what got in the way. Help identify whether the commitment
   was realistic.

6. **Adjust or recommit.** Help the user either adjust the goal
   to something achievable or recommit with a new timeline. Record
   the updated commitment.

## Output

After each accountability check, write a structured entry to the
journal:

```
## Accountability Check - [date]

**Commitment:** [original commitment with date]
**Status:** [completed | partially completed | not started | adjusted]
**Details:** [specific evidence or user's report]
**Next step:** [new commitment or acknowledgment of completion]
```

Record accountability check results in the journal every time
this skill activates, even if the user completed the commitment
successfully. The record creates a pattern that is useful for
future coaching.
